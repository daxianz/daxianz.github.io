免费 GPU 算力平台整理

学生党，整理了一些用过的免费 GPU 算力平台。

百度飞桨 AI Studio

网址： 百度飞桨 AI Studio

获取方式

  • 每日运行项目赠送 8 点 GPU 免费额度
  • 赠送的 8 点只有 48 小时有效期

GPU 使用入口

创建项目 → 运行 → 选择运行环境

注意事项

  • 使用异构算力需要在创建项目时，将“项目架构”选择为“异构框架”
  • 免费 GPU 资源有限，可能需要排队
  • 真不好用

部分算力配置

GPU Iluvatar BI-V150S
显存 32GB
CPU 4 核
内存 32GB
硬盘 100GB

Kaggle

网址： Kaggle Code

获取方式

  • 每周固定 30 小时算力资源
  • 每周自动重置

GPU 使用入口

Home → Notebooks → Create

注意事项

  • 必须使用手机号验证账号后才能开启 GPU 权限
  • 手机验证问题参考：
    Kaggle 成功验证手机号方法（知乎）； 也可以尝试在手机 Chrome 中登录 Kaggle 进行验证

腾讯云 Cloud Studio

网址： 腾讯云 Cloud Studio

获取方式

  • 绑定腾讯云账号赠送 50 小时机时
  • 每日签到赠送 2 机时，每日签到赠送的机时 7 天过期
  • 购买价格：1 机时 = 1 元，最低购买量：10 机时

GPU 使用入口

创建应用 → 编辑代码 → 切换算力类型

注意事项

  • 所有配置均固定提供 256GB 硬盘

阿里云 PAI

参考文档

  • 阿里云免费试用平台
  • PAI 免费试用说明

额度与有效期：一次性消耗式抵扣包，用完为止，总计750计算时 (CU-Hour)，有效期3个月。
分配方式：每月发放250计算时。自领取日起，首个与最后一个非完整月度，其额度合并计算，其余完整月份额度当月有效，未用完的额度月末作废。

OpenI 启智

网址： OpenI 启智

有很多算力可以尝试，NVIDIA的GPU，昇腾NPU，还有燧原GCU、寒武纪MLU、海光DCU、天数智芯GPGPU、沐曦GPGPU。每天可以做任务领取积分，一般情况下每天做满常规任务得到29积分，最简单的是提交一个任务马上领取10积分。感觉想要使用里面的算力资源还是特别好攒的。里面的T4是可以免费使用的。

Lightning AI

网址： Lightning AI

国外平台，每个月会给15美元额度，仅当月可用，里面有一个非常便宜的AWS的T4（16GB），0.19刀每小时，如果只用这个可以大概使用80h；还有L4（24GB），0.48刀每小时，可以使用20h；如果允许抢占，L40S可以1.64刀每小时，可以使用9个小时。其他的太贵就根本用不起的。
